# Master Social Media Management Workspace

Welcome to the **Master Social Media Management** console. This workspace provides a unified dashboard to manage the social media profiles, calendars, voice guidelines, and access credentials for the three core businesses: **Corely**, **Suriosity**, and **Lansem**.

---

## 📂 Workspace Structure

The workspace is structured as a premium, client-side single-page application (SPA) built using HTML5, Vanilla CSS, and native JavaScript:

*   **[`index.html`](file:///c:/Users/ARNAB/.gemini/antigravity-ide/scratch/Master%20social%20media%20management/index.html)**: The main layout structure including navigation sidebars, dynamic views (Dashboard, Account Matrix, Scheduler, Guidelines), and popup modals.
*   **[`styles.css`](file:///c:/Users/ARNAB/.gemini/antigravity-ide/scratch/Master%20social%20media%20management/styles.css)**: A premium styling system implementing responsive flexbox/grid grids, modern dark mode, glassmorphic card overlays, and dynamic brand accents.
*   **[`app.js`](file:///c:/Users/ARNAB/.gemini/antigravity-ide/scratch/Master%20social%20media%20management/app.js)**: State controller managing brand selections, rendering SVG analytics line graphs, managing local storage post scheduling, and credentials retrieval.

## 🚀 How to Run the Dashboard

Since SMM Master is built as an interactive client-side app, you can launch it instantly:

1.  Locate `index.html` in your file explorer.
2.  Double-click it to open in any modern web browser (Chrome, Edge, Firefox, Safari).
3.  *Note:* The scheduler is fully persistent. Any scheduled or deleted posts will remain saved in your browser's `localStorage` even after refreshing.
