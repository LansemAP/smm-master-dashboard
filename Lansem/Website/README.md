# LANSEM - Legal & Accounting Advisors Website

A premium, highly responsive corporate website for LANSEM, delivering qualified legal and accounting back-office support for law firms, accounting practices, and corporate teams in the UK, US, and UAE.

## Features
- **Modern Responsive Design**: Clean grid layouts, fluid mobile navigation, and modern typography using the Plus Jakarta Sans font.
- **Dynamic Hero Slideshow**: Automatic background transitions synced with GSAP-animated, slide-dependent value proposition headlines.
- **Interactive Proximity CTA Buttons**: Premium GreenSock GSAP-driven magnetic attraction animations on cursor hover.
- **High-Performance Scroll Reveals**: Native browser-optimized `IntersectionObserver` scroll entrance reveals for section headers, grids, and individual columns (eliminating coordinate calculation delays and layout shifts).
- **Interactive FAQ Accordion**: Fully functional accordion for user queries.

## Tech Stack
- HTML5 (Semantic markup)
- CSS3 (Vanilla design tokens, CSS custom properties, responsive flexbox/grid systems)
- JavaScript (ES6+ Vanilla)
- GreenSock GSAP (for core timeline and button animations)
